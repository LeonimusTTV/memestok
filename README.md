# MemesTok

A desktop app made to watch memes from Reddit, built with Tauri, React, and TypeScript.

## Prerequisites

- [Bun](https://bun.sh)
- [Rust](https://www.rust-lang.org/tools/install)

## Install

```bash
bun install
```

## Run

```bash
bun run tauri dev
```

## Build

```bash
bun run build:app
```
